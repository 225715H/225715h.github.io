タイトルの描き方

以下のようタイトルについてそれぞれ機能を説明する。

  • title: ポストのタイトルを反映する

  • lead: タイトルの下のリード文的なのを反映する

  • description: よくわかっていない。

  • thumbnail: サイトのタブに画像の挿入ができる。visibility: list でリストでの画像表示ができる。指定しなければポストでも表示される。

  • date: 日付が反映される

  • tags: タグ付けができる

  • categories: カテゴリー分けができる

  • menu: 上部にあるメニューバーみたいなところに設定すると追加できる。 weight:で置く位置を指定できる（多分）。 name:でメニューバーに反映される名前を決められる。指定しない場合、タイトルと同じになる。

    weight:

マークダウンの書き方

基本はthemes/Mainroad/sampleSite/post/basic-elements.mdに書いてある。

カスタマイズ

config.tomlの設定

baseurl = "/"
languageCode = "en-us"
paginate = "10" # Number of posts per page
theme = "mainroad"
disqusShortname = "" # Enable comments by entering your Disqus shortname
googleAnalytics = "" # Enable Google Analytics by entering your tracking id
hasCJKLanguage = true # 日本語・中国語・韓国語の単語カウントを有効にする
summaryLength = 50 # サマリーの長さを120単語にする

[Author]
  name = "Motoki"
  bio = "I'm a student."
  avatar = "img/ava.png"

[Params]
  description = "John Doe's Personal blog about everything" # Description of your site
  opengraph = true
  twitter_cards = false
  readmore = false # Show "Read more" button in list if true
  authorbox = true
  pager = true
  post_meta = ["date", "categories"] # Order of post meta information
  mainSections = ["post", "docs"]

[Params.logo]
  title = "My Site."
  subtitle = "Portfolio." # Logo subtitle
  image = "img/logo.png"

[Params.style.vars]
  highlightColor = "#007BA7"


[Params.sidebar]
  home = "right"   # Configure layout for home page
  list = "right"   # Configure layout for list pages
  single = "right" # Configure layout for single pages
  # Enable widgets in given order
  widgets = ["search", "recent", "taglist"]

[Params.widgets]
  recent_num = 5 # Set the number of articles in the "Recent articles" widget
  categories_counter = true # Enable counter for each category in "Categories" widget
  tags_counter = true # Enable counter for each tag in "Tags" widget (disabled by default)


[Params.thumbnail]
  # Show thumbnail only for list items
  visibility = ["list"]

# メニューバーの設定
[[Menus.main]]
  Name = "posts"
  URL = "/" # content配下のディレクトリを指定することでそのディレクトリ内のポストだけ表示できる
  weight = 1

ポスト内の画像

static配下に任意のディレクトリを作成し、そこに画像を置く。

画像を置きたいポストの任意の行で![キャプション名](/<static>配下のディレクトリ名/ファイル名)を書くことで画像が反映される。
