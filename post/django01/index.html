Webアプリのバージョンアップ

ローカルで変更した内容をgitで更新する。

git add .
git commit -m "message"
git push origin main

リモートに移動してgitの変更を反映させる。

git pull origin main

djangoでアプリ開発をしたのでプロジェクトに移動して、元々ある仮想環境を立ち上げた後に、migrateする。 ちなみに、migrateはmodels.pyのデータベース設定を反映させるコマンドである。

source todoenv/bin/activate
python3 manage.py makemigrations
python3 manage.py migrate
