ESLintの導入

ESLintは、コードの書き方を提案してくれる。nvs use ltsでnpmコマンドを使えるようにする。

以下に従って、インストールとeslint の設定を行う。

npm install eslint --save-dev
npx eslint --init
You can also run this command directly using 'npm init @eslint/config'.

✔ Which framework does your project use? · react  # reactを選択

✔ Would you like to install them now? · No / Yes  # yesを選択

vsCodeの拡張機能でESLintもインストールし、有効にしておく。

インストールした後に作成された.eslintrs.jsonのルールに以下を追加する。

"rules": {
        "react/react-in-jsx-scope": 0
    }
