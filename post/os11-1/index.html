## console から UEFI shell 経由で DVD boot する
+braun+e225715 ie-virsh console 02

UEFI の Shell> を exitで抜けると、UEFI manu がでる ここで、boot manager menu で QEMU DVD-ROM を選択する その後、install-fedora を選択する control + ]　で抜ける
